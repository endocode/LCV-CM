Example of usage of the API methods /LicensesInput and /LicensesInputSPDX

  • http://0.0.0.0:8080/LicensesInput?InboundLicenses=Apache 2.0,MIT License&OutboundLicense=MIT
  • http://0.0.0.0:8080/LicensesInputSPDX?InboundLicenses=Apache-2.0,MIT&OutboundLicense=MIT

These two endpoints accept both POST and GET requests, and they can be used via browser or collaboration platform for API development, such as Postman.

The syntax is as following:

  • /LicensesInput?InboundLicenses=FirstInboundLicense,FirstInboundLicense, ... nInboundLicense&OutboundLicense=UniqueOutboundLicenseSPDXid

  • /LicensesInputSPDX?InboundLicenses=FirstInboundLicenseSPDXid,FirstInboundLicenseSPDXid, ... nInboundLicenseSPDXid&OutboundLicense=UniqueOutboundLicenseSPDXid
